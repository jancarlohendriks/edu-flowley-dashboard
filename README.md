## To run this project

Fill in ENV

```
HOST= // localhost:3000
GSA_CLIENT_EMAIL= // Google Service Account for GA4
GSA_PRIVATE_KEY= // Google Service Account for GA4
FIREBASE_PROJECT_ID=
FIREBASE_PRIVATE_KEY=
FIREBASE_CLIENT_EMAIL=
FIREBASE_DB_URL=
```

Private keys have unix "\n" new lines, not needed on Vercel.

```
npm i
```

Then

```
npm run dev
```
